# aPaaS 元数据驱动平台体系建设 — 方案设计

> 版本：v2.0 | 日期：2026-03-31
> 前置文档：[需求澄清](产品-需求澄清.md)
> 需求类型：`架构重构` | 特征标签：`数据模型变更`、`跨租户/权限`

## 一句话方案
基于元模型驱动的三层架构（元模型定义→元数据实例→业务数据），通过大宽表 Schema-on-Read 机制实现业务对象声明式配置，Common/Tenant 双层合并实现多租户隔离与出厂数据独立升级。

## 方案设计

### 扩展策略
**选择**：改造式扩展
**理由**：
1. 新能力改变了元数据的底层存储和读取方式（从硬编码表到大宽表 Schema-on-Read）
2. 不使用新能力时，老系统的硬编码方式无法继续工作（编码不一致、ID 关联断裂）
3. 配置入口：metarepo-web 管理前端 + MetamodelBrowseApiService 内部 API

**对现有功能的影响**：
- entity-service：需适配新的元数据读取接口（MetaRepoReadApi 替代老接口）
- layout-service：字段定义来源从硬编码改为元数据查询
- rule-service：checkRule 从独立表改为大宽表合并读取
- 数据迁移期间需保证老接口兼容

### 核心流程

**元数据读取流程（Common/Tenant 合并）：**
| 步骤 | 动作 | 响应 | 异常处理 |
|:---|:---|:---|:---|
| 1 | 接收请求（tenantId + metamodelApiKey + entityApiKey） | — | 参数缺失返回 400 |
| 2 | 查询 MetaModel 定义（enable_common / enable_tenant / db_table） | 获取存储配置 | 元模型不存在返回 404 |
| 3 | if enable_common=1：查 p_common_metadata + CommonMetadataConverter 转换 | 获取 Common 级业务 Entity 列表 | 查询超时降级返回空列表 |
| 4 | if enable_tenant=1：查 db_table 指向的 Tenant 表 | 获取 Tenant 级数据 | 查询超时降级返回空列表 |
| 5 | MergeReadService 合并（Common有Tenant无→用Common，同apiKey→Tenant覆盖，delete_flg=1→隐藏） | 合并结果 | — |
| 6 | namespace=product 检查 license | 过滤无权限数据 | license 校验失败隐藏该条 |
| 7 | 返回业务 Entity 列表 | 响应 | — |

**元数据写入流程：**
| 步骤 | 动作 | 响应 | 异常处理 |
|:---|:---|:---|:---|
| 1 | 接收写入请求（tenantId + 业务 Entity） | — | 参数校验失败返回 400 |
| 2 | Schema 校验（p_meta_option 取值范围 + p_meta_item 必填/唯一） | 校验通过 | 校验失败返回具体字段错误信息 |
| 3 | CommonMetadataConverter.toTenantMetadata() 转换 | 生成 TenantMetadata | 转换异常返回 500 |
| 4 | DynamicTableNameHolder.executeWith(dbTable) 路由写入 | INSERT/UPDATE | 写入失败事务回滚 |
| 5 | if enable_log=1：写入 p_tenant_meta_log | 记录变更日志 | 日志写入失败不影响主流程 |
| 6 | 返回写入结果 | 成功响应 | — |

**遮蔽删除流程（删除 Common 级数据）：**
| 步骤 | 动作 | 响应 | 异常处理 |
|:---|:---|:---|:---|
| 1 | 判断目标数据来源（Common / Tenant） | — | — |
| 2 | if Tenant 已有记录：软删除（update delete_flg=1） | 标记删除 | — |
| 3 | if 仅 Common 有：插入 delete_flg=1 的 Tenant 记录 | 遮蔽 Common 数据 | — |
| 4 | if 有级联子元数据（p_meta_link.cascade_delete=1）：递归删除 | 级联处理 | 级联失败整体回滚 |

### 权限设计
| 角色 | 元模型管理 | Common 元数据 | Tenant 元数据 | 管理前端 |
|:---|:---:|:---:|:---:|:---:|
| 平台管理员 | 读写 | 读写（初始化/Module安装） | 读写 | 全部功能 |
| 租户管理员 | 只读 | 只读（合并读取） | 读写 | 元数据浏览+编辑 |
| 业务开发者 | 只读 | 只读（通过 API） | 只读（通过 API） | 元模型浏览 |
| 普通用户 | 无 | 无 | 无 | 无 |

**namespace 权限控制：**
- system：所有租户可见，不受 license 控制
- product：受 license 控制，付费租户可见
- custom：仅该租户管理员可见
