## Tab 1：基本信息

编辑 entity 元数据本身的属性：

| 字段 | 类型 | 说明 |
|:-----|:-----|:-----|
| label | 文本 | 业务对象名称 |
| apiKey | 文本（只读） | API 名称 |
| description | 文本域 | 描述 |
| svgApiKey | 图标选择器 | 对象图标 |
| entityType | 只读标签 | 标准/自定义/系统 |
| namespace | 只读标签 | system/product/custom |
| 启用开关组 | 复选框组 | 以下开关 |

启用开关（参考老系统弹框）：

| 开关 | 字段 | 说明 |
|:-----|:-----|:-----|
| 启用操作日志 | historyLogFlg | 自动记录操作记录 |
| 启用业务类型 | busiTypeFlg | 添加自定义业务类型 |
| 启用校验规则 | checkRuleFlg | 自定义校验规则 |
| 启用查重规则 | duplicateRuleFlg | 自定义查重规则 |
| 启用业务逻辑代码 | scriptExecutorFlg | 执行业务逻辑代码 |
| 启用团队成员 | groupMemberFlg | 团队成员功能（不可关闭） |
| 启用系统动态流 | dynamicFeedFlg | 自动发送动态流 |
| 启用共享 | sharingFlg | 共享规则 |
| 启用流程 | flowFlg | 审批流程 |

## Tab 2：字段管理

管理 entity 下的 item（字段）列表：

| 功能 | 说明 |
|:-----|:-----|
| 字段列表 | 表格展示：字段名、apiKey、类型、必填、启用状态、排序 |
| 新建字段 | 弹框：字段名、apiKey、类型（下拉）、描述、必填、默认值 |
| 编辑字段 | 弹框：修改字段属性 |
| 启用/禁用 | 行操作 |
| 排序 | 拖拽或上下箭头 |
| 字段类型 | TEXT/SELECT/NUMBER/DATE/BOOLEAN/PHONE/EMAIL/URL/... |
| 选项值管理 | SELECT/MULTI_SELECT 类型字段的选项值（pickOption 子表） |

## Tab 3：校验规则

管理 entity 下的 checkRule 列表：

| 功能 | 说明 |
|:-----|:-----|
| 规则列表 | 表格：规则名、公式、错误提示、激活状态 |
| 新建规则 | 弹框：规则名、校验公式、错误提示、关联字段 |
| 编辑规则 | 弹框 |
| 激活/停用 | 行操作 |
| 删除 | 行操作 |

## Tab 4：查重规则

管理 entity 下的 duplicateRule 列表：

| 功能 | 说明 |
|:-----|:-----|
| 规则列表 | 表格：规则名、条件逻辑、启用状态 |
| 新建规则 | 弹框：规则名、条件字段选择、匹配方式、逻辑组合 |
| 编辑规则 | 弹框 |
| 启用/停用 | 行操作 |
| 查重条件 | 子表：选择字段 + 匹配方式（精确/模糊/包含） |

## Tab 5：字段组

管理 entity 下的 fieldSet 列表：

| 功能 | 说明 |
|:-----|:-----|
| 字段组列表 | 表格：组名、包含字段数 |
| 新建字段组 | 弹框：组名、描述 |
| 编辑字段组 | 弹框 |
| 字段组成员 | 子表：从 item 列表中勾选字段加入组 |
