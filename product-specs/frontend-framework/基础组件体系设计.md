# 基础组件体系设计

> 版本：1.0 | 日期：2026-04-18
> 目标：将画布组件拆解为最基础的原子组件，用户可自由组合配置

---

## 一、当前问题

"选项集管理"是一个大黑盒组件，内部包含标题、描述、新增按钮、数据表格、行操作按钮。用户无法：
- 单独调整标题的文字和样式
- 选择表格显示哪些列
- 决定操作按钮的位置和类型
- 在表格上方加搜索框
- 调整各部分的排列顺序

## 二、设计原则

**原子化**：每个组件只做一件事，通过组合实现复杂页面。

```
当前：
  [选项集管理] = 标题 + 描述 + 新增按钮 + 表格 + 行操作（一个大组件）

目标：
  [页面标题]     ← 可配置文字、描述、图标
  [操作按钮组]   ← 可配置按钮列表（新增、导入、导出）
  [数据表格]     ← 可配置列、数据源、分页、行操作
  [搜索栏]       ← 可配置搜索字段
```

## 三、基础组件清单

### 3.1 布局组件

| 组件 | 说明 | 可配置项 |
|:---|:---|:---|
| 行容器 (Row) | 水平排列子组件 | 间距、对齐方式 |
| 列容器 (Column) | 垂直排列子组件 | 宽度比例（1/2、1/3、1/4） |
| 卡片 (Card) | 带边框和标题的容器 | 标题、是否可折叠 |
| 分隔线 (Divider) | 水平分隔线 | 标题文字 |
| 间距 (Spacer) | 空白间距 | 高度 |

### 3.2 展示组件

| 组件 | 说明 | 可配置项 |
|:---|:---|:---|
| 页面标题 (PageTitle) | 标题 + 描述 | 标题文字、描述文字、图标 |
| 统计数字 (StatNumber) | 大数字 + 标签 | 标签、数据源、格式 |
| 文本块 (TextBlock) | 纯文本/富文本 | 内容、字号、颜色 |
| 图标 (Icon) | 图标展示 | 图标名、大小、颜色 |

### 3.3 数据组件

| 组件 | 说明 | 可配置项 |
|:---|:---|:---|
| 数据表格 (DataTable) | 表格展示 | 数据源、列定义、分页、行操作 |
| 数据列表 (DataList) | 列表展示 | 数据源、列表项模板 |
| 树形列表 (TreeView) | 树形展示 | 数据源、标签字段、父级字段 |
| 表单 (Form) | 表单输入 | 数据源、字段列表、列数、模式 |
| 详情面板 (DetailPanel) | 只读详情 | 数据源、字段列表 |

### 3.4 交互组件

| 组件 | 说明 | 可配置项 |
|:---|:---|:---|
| 按钮 (Button) | 单个按钮 | 文字、图标、类型（主要/次要/危险）、点击动作 |
| 按钮组 (ButtonGroup) | 多个按钮 | 按钮列表、排列方式 |
| 搜索框 (SearchInput) | 搜索输入 | 占位文字、搜索字段 |
| 筛选器 (Filter) | 下拉筛选 | 筛选字段、选项来源 |
| 标签页 (Tabs) | 多标签切换 | 标签列表，每个标签是一个 DropZone |

### 3.5 字段组件（用于 Form 内部）

| 组件 | 对应 ItemType | 可配置项 |
|:---|:---|:---|
| 文本输入 (TextInput) | TEXT | 占位文字、最大长度 |
| 下拉选择 (Select) | SELECT | 选项来源、是否多选 |
| 日期选择 (DatePicker) | DATE | 格式、范围 |
| 数字输入 (NumberInput) | NUMBER | 步长、范围 |
| 开关 (Switch) | BOOLEAN | 标签 |
| 文件上传 (FileUpload) | FILE/IMAGE | 文件类型、大小限制 |

## 四、组合示例 — 语言管理页面

用户在画布中自由组合基础组件：

```
┌─────────────────────────────────────────────────┐
│ [PageTitle]                                      │
│   标题: "语言管理"                                │
│   描述: "管理系统支持的语言列表"                   │
│   图标: Globe                                    │
├─────────────────────────────────────────────────┤
│ [Row]                                            │
│   ├── [SearchInput] 搜索语言...                  │
│   └── [Button] + 新增选项  (动作: 打开新增弹框)   │
├─────────────────────────────────────────────────┤
│ [DataTable]                                      │
│   数据源: globalPickOption / language             │
│   列:                                            │
│     - 序号 (自动)                                │
│     - 值(apiKey) (text)                          │
│     - 显示标签(label) (text)                     │
│     - 默认(defaultFlg) (tag)                     │
│     - 状态(enableFlg) (tag)                      │
│   行操作:                                        │
│     - [Button] 编辑 (动作: 打开编辑弹框)          │
│     - [Button] 启用/禁用 (动作: API调用)          │
│     - [Button] 删除 (动作: 确认后API调用)         │
│   分页: 每页 20 条                               │
└─────────────────────────────────────────────────┘
```

### 与当前"选项集管理"大组件的对比

| 维度 | 大组件方式 | 基础组件方式 |
|:---|:---|:---|
| 标题 | 固定在组件内部 | 独立 PageTitle 组件，可自由配置 |
| 新增按钮 | 固定在右上角 | 独立 Button 组件，可放任意位置 |
| 搜索 | 没有 | 可添加 SearchInput 组件 |
| 表格列 | 通过属性配置 | DataTable 组件，列可拖拽排序 |
| 行操作 | 通过属性配置 | DataTable 内的 Button 组件 |
| 布局 | 固定 | Row/Column 自由排列 |

## 五、Puck 中的实现

### 5.1 DropZone 嵌套

Puck 支持 DropZone，允许组件内部嵌套子组件：

```typescript
// Row 容器组件 — 内部有一个 DropZone
const Row = {
  label: '行容器',
  defaultProps: { gap: 16, align: 'center' },
  fields: {
    gap: { type: 'number', label: '间距(px)' },
    align: { type: 'select', label: '对齐', options: [...] },
  },
  render: ({ gap, align, puck: { renderDropZone } }) => (
    <div style={{ display: 'flex', gap, alignItems: align }}>
      {renderDropZone({ zone: 'row-content' })}
    </div>
  ),
};

// Column 容器 — 指定宽度比例
const Column = {
  label: '列容器',
  defaultProps: { width: '1/2' },
  fields: {
    width: { type: 'select', label: '宽度', options: [
      { label: '1/4', value: '25%' },
      { label: '1/3', value: '33.33%' },
      { label: '1/2', value: '50%' },
      { label: '2/3', value: '66.67%' },
      { label: '全宽', value: '100%' },
    ]},
  },
  render: ({ width, puck: { renderDropZone } }) => (
    <div style={{ width }}>
      {renderDropZone({ zone: 'column-content' })}
    </div>
  ),
};
```

### 5.3 Button 组件

```typescript
const Button = {
  label: '按钮',
  defaultProps: {
    text: '按钮',
    icon: '',
    variant: 'primary',
    actionType: 'none',
    actionTarget: '',
  },
  fields: {
    text: { type: 'text', label: '按钮文字' },
    icon: { type: 'text', label: '图标名' },
    variant: {
      type: 'select',
      label: '样式',
      options: [
        { label: '主要', value: 'primary' },
        { label: '次要', value: 'secondary' },
        { label: '危险', value: 'danger' },
        { label: '文字', value: 'text' },
      ],
    },
    actionType: {
      type: 'select',
      label: '点击动作',
      options: [
        { label: '无', value: 'none' },
        { label: '打开弹框', value: 'openDialog' },
        { label: '调用接口', value: 'apiCall' },
        { label: '跳转页面', value: 'navigate' },
        { label: '刷新数据', value: 'refresh' },
      ],
    },
    actionTarget: { type: 'text', label: '动作目标（弹框apiKey / 接口地址）' },
  },
  render: ({ text, variant }) => (
    <button style={{
      padding: '6px 16px',
      borderRadius: 8,
      fontSize: 13,
      fontWeight: 500,
      background: variant === 'primary' ? '#2563eb' : variant === 'danger' ? '#dc2626' : '#f3f4f6',
      color: variant === 'primary' || variant === 'danger' ? '#fff' : '#374151',
      border: 'none',
      cursor: 'default',
    }}>
      {text}
    </button>
  ),
};
```

## 六、实现路径

### Phase 1：布局 + 展示组件（1 周）

- Row / Column / Card / Divider / Spacer
- PageTitle / TextBlock
- Button / ButtonGroup
- 用 Puck DropZone 实现嵌套

### Phase 2：数据组件（2 周）

- DataTable（核心，支持列定义、分页、行操作）
- SearchInput / Filter
- TreeView

### Phase 3：表单组件（2 周）

- Form 容器
- 字段组件（TextInput / Select / DatePicker / NumberInput / Switch）
- 字段从元数据自动推导

### Phase 4：动作系统（1 周）

- Button 的 actionType 执行器
- 打开弹框 / 调用 API / 跳转页面 / 刷新数据
- 与 pageAction 元模型对接
