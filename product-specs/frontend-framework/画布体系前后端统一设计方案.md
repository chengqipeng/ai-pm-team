> 版本：1.0 | 日期：2026-04-20
> 定位：画布编辑器 + 后端元模型 + 元数据实例 + API 接口的完整闭环体系

## 一、体系全景


## 二、核心元模型定义

### 2.1 page（页面定义）

已注册，字段：

| api_key | label | 说明 |
|:---|:---|:---|
| pageName | 页面名称 | 显示名 |
| pageType | 页面类型 | 1=全屏 2=弹框 3=抽屉 4=内嵌 |
| entityApiKey | 主实体 | 绑定的业务实体/元模型 |
| pageDescription | 描述 | |
| dialogWidth | 弹框宽度 | 仅 pageType=2/3 |
| layoutColumns | 表单列数 | 1/2 |
| enableFlg | 启用 | |

### 2.2 pageSection（页面区域）

已注册，字段：

| api_key | label | 说明 |
|:---|:---|:---|
| sectionKey | 区域标识 | canvas/header/sidebar |
| componentType | 组件类型 | CanvasSchema（画布 JSON） |
| componentProps | 组件配置 | **画布 JSON Schema 存储在此** |
| sectionOrder | 排序 | |
| layoutMode | 布局模式 | 1=整行 2=半行 3=三分之一 |

### 2.3 pageAction（页面动作定义）— 新增

**这是关键的新增元模型**，定义每个页面上的按钮点击后做什么。

```sql
INSERT INTO p_meta_model (api_key, label, namespace, enable_common, enable_tenant, db_table)
VALUES ('pageAction', 'UI页面动作', 'system', 1, 1, 'p_tenant_page_action');
```

字段定义：

| api_key | label | item_type | db_column | 说明 |
|:---|:---|:---:|:---|:---|
| actionKey | 动作标识 | 1(TEXT) | dbc_varchar1 | 页面内唯一，如 create/edit/delete |
| actionType | 动作类型 | 2(SELECT) | dbc_varchar2 | 对应 uiAction.apiKey |
| targetPageApiKey | 目标页面 | 1(TEXT) | dbc_varchar3 | create/edit 打开哪个弹框页面 |
| targetMetamodelApiKey | 目标元模型 | 1(TEXT) | dbc_varchar4 | 数据操作的目标元模型 |
| targetEntityApiKey | 目标分组 | 1(TEXT) | dbc_varchar5 | 数据操作的目标分组 |
| buttonStyle | 按钮样式 | 1(TEXT) | dbc_varchar6 | 对应 uiButtonStyle.apiKey |
| buttonLabel | 按钮文字 | 1(TEXT) | dbc_varchar7 | 显示文字 |
| confirmText | 确认文案 | 1(TEXT) | dbc_varchar8 | 需要确认时的提示文字 |
| actionOrder | 排序 | 5(NUMBER) | dbc_int1 | |
| enableFlg | 启用 | 31(BOOLEAN) | dbc_smallint1 | |
| placement | 位置 | 2(SELECT) | dbc_varchar9 | toolbar/row/form（工具栏/行操作/表单） |

### 2.4 关联关系

```
page (1) ──→ (N) pageSection    通过 entityApiKey 关联
page (1) ──→ (N) pageAction     通过 entityApiKey 关联
pageAction.targetPageApiKey ──→ page.apiKey   弹框页面引用
pageAction.actionType ──→ uiAction.apiKey     动作类型引用
pageAction.buttonStyle ──→ uiButtonStyle.apiKey 样式引用
```

## 三、语言管理完整实例

### 3.1 page 实例

| apiKey | pageName | pageType | 说明 |
|:---|:---|:---:|:---|
| pageSettingsLang | 语言管理 | 1(全屏) | 语言列表页面 |
| pageForm_globalPickOption | 新增/编辑选项 | 2(弹框) | 选项表单弹框 |

### 3.2 pageSection 实例

| apiKey | entityApiKey(所属页面) | componentType | 说明 |
|:---|:---|:---|:---|
| pageSettingsLang_canvas | pageSettingsLang | CanvasSchema | 语言列表画布 |
| pageForm_gpo_canvas | pageForm_globalPickOption | CanvasSchema | 选项表单画布 |

### 3.3 pageAction 实例

| apiKey | entityApiKey(所属页面) | actionKey | actionType | targetPageApiKey | buttonStyle | buttonLabel | placement |
|:---|:---|:---|:---|:---|:---|:---|:---|
| langAction_create | pageSettingsLang | create | create | pageForm_globalPickOption | primary | 新增语言 | toolbar |
| langAction_refresh | pageSettingsLang | refresh | refresh | | default | 刷新 | toolbar |
| langAction_edit | pageSettingsLang | edit | edit | pageForm_globalPickOption | link | 编辑 | row |
| langAction_delete | pageSettingsLang | delete | delete | | danger | 删除 | row |
| gpoForm_save | pageForm_globalPickOption | save | save | | primary | 保存 | form |
| gpoForm_cancel | pageForm_globalPickOption | cancel | cancel | | default | 取消 | form |

### 3.4 画布 JSON Schema

语言列表画布不再硬编码按钮的 action，而是引用 pageAction：

```json
{
  "version": "1.0",
  "components": [
    {"id": "lang_title", "type": "title", "props": {"text": "语言管理"}},
    {"id": "lang_desc", "type": "text", "props": {"content": "管理系统支持的语言列表..."}},
    {"id": "lang_spacer", "type": "spacer", "props": {"height": 12}},
    {"id": "lang_toolbar", "type": "actionBar", "props": {"placement": "toolbar"}},
    {"id": "lang_table", "type": "dataTable", "props": {
      "metamodelApiKey": "globalPickOption",
      "entityApiKey": "language",
      "columns": ["optionOrder", "apiKey", "label", "defaultFlg", "enableFlg"],
      "rowActions": "auto"
    }}
  ]
}
```

注意：
- `actionBar` 组件自动从 pageAction 加载 placement=toolbar 的按钮
- `dataTable` 的 `rowActions: "auto"` 自动从 pageAction 加载 placement=row 的按钮
- 按钮的样式、文字、事件全部从 pageAction 元数据读取

## 四、前端运行时加载流程

```
用户点击菜单"语言管理"
    ↓
PageRuntime(pageApiKey='pageSettingsLang')
    ↓ 并行加载
    ├── listMetadata('page') → 找到页面定义
    ├── listMetadata('pageSection', 'pageSettingsLang') → 找到画布 Schema
    └── listMetadata('pageAction', 'pageSettingsLang') → 找到 6 个动作定义
    ↓
渲染画布：
    ├── title → "语言管理"
    ├── text → 说明文字
    ├── spacer → 间距
    ├── actionBar → 从 pageAction 中过滤 placement=toolbar
    │   → 渲染"新增语言"(primary) + "刷新"(default)
    └── dataTable → 加载数据 + 从 pageAction 过滤 placement=row
        → 每行渲染"编辑"(link) + "删除"(danger)
    ↓
用户点击"新增语言"
    ↓
ActionContext.dispatch
    ↓ 从 pageAction 找到 langAction_create
    ↓ actionType=create, targetPageApiKey=pageForm_globalPickOption
    ↓
CanvasModal(pageApiKey='pageForm_globalPickOption')
    ↓ 并行加载
    ├── listMetadata('pageSection', 'pageForm_globalPickOption') → 表单画布
    └── listMetadata('pageAction', 'pageForm_globalPickOption') → 保存/取消
    ↓
渲染弹框画布：
    ├── fieldText(apiKey) → 绑定 formData
    ├── fieldText(label) → 绑定 formData
    ├── fieldNumber(optionOrder) → 绑定 formData
    ├── fieldBoolean(defaultFlg) → 绑定 formData
    ├── fieldBoolean(enableFlg) → 绑定 formData
    └── actionBar(placement=form) → "取消" + "保存"
    ↓
用户填写表单，点击"保存"
    ↓ 从 pageAction 找到 gpoForm_save
    ↓ actionType=save
    ↓ submitForm(formData) → batchSaveMetadata
    ↓ 成功 → 关闭弹框 → 父页面 refreshKey++
```

## 五、后端需要做的事

### 5.1 注册 pageAction 元模型的 p_meta_item

11 个字段（见 2.3 节），写入 p_meta_item。

### 5.2 写入 Common 级 pageAction 种子数据

语言管理 6 个动作 + 时区管理 6 个动作 + 选项表单 2 个动作。

### 5.3 API 无需改动

现有的 `GET /meta/metadata?metamodelApiKey=pageAction&entityApiKey=pageSettingsLang` 已经能正确返回。

### 5.4 后续：每新增一个管理页面

1. 在 page 中新增一条记录（页面定义）
2. 在 pageSection 中新增画布 Schema
3. 在 pageAction 中新增该页面的按钮动作定义
4. 如果有弹框 → 在 page 中新增弹框页面 + pageSection 画布 + pageAction 动作

## 六、前端需要新增的组件

### 6.1 ActionBar 原子组件

自动从 pageAction 加载按钮，按 placement 过滤：

```typescript
const actionBarAtom: AtomDef = {
  type: 'actionBar',
  props: { placement: 'toolbar' },  // toolbar / row / form
  render: (props, ctx) => {
    // 从 ctx.pageActions 中过滤 placement 匹配的动作
    // 按 actionOrder 排序
    // 渲染按钮（样式从 uiButtonStyle 读取，文字从 pageAction.buttonLabel 读取）
  }
};
```

### 6.2 DataTable 的 rowActions

dataTable 组件的 `rowActions: "auto"` 模式：
- 从 pageAction 中过滤 placement=row 的动作
- 每行渲染对应的操作按钮

## 七、与现有体系的关系

| 层级 | 已有 | 新增 | 说明 |
|:---|:---|:---|:---|
