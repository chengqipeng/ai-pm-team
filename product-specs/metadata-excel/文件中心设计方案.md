# 文件中心设计方案

## 1. 核心思路

```
之前：业务方上传 → 拿到 path/url → 自己存 path → 自己拼 url
之后：业务方上传 → 文件中心存文件+记录元数据 → 返回 fileId → 业务方只存 fileId → 用 fileId 查一切
```

## 2. 文件元数据表

```sql
CREATE TABLE p_file_meta (
    id              BIGINT NOT NULL,
    delete_flg      SMALLINT NOT NULL DEFAULT 0,
    created_at      BIGINT NOT NULL,
    created_by      BIGINT NOT NULL,
    updated_at      BIGINT NOT NULL,
    updated_by      BIGINT NOT NULL,

    tenant_id       BIGINT NOT NULL,
    file_name       VARCHAR(512) NOT NULL,      -- 原始文件名
    file_ext        VARCHAR(32),                -- 扩展名（jpg/pdf/xlsx）
    file_size       BIGINT NOT NULL DEFAULT 0,  -- 字节数
    content_type    VARCHAR(128),               -- MIME 类型
    file_group      VARCHAR(64) NOT NULL,       -- 分组：image/doc/video/audio/temp
    storage_path    VARCHAR(1024) NOT NULL,      -- OSS 相对路径
    storage_type    VARCHAR(32) NOT NULL,        -- 存储类型：local/s3/ali/cos 等
    access_url      VARCHAR(1024),              -- 公网访问 URL（public 文件）
    md5             VARCHAR(64),                -- 文件 MD5（可选，用于去重）
    biz_type        VARCHAR(128),               -- 业务类型标识（avatar/attachment/excel-export 等）
    biz_id          VARCHAR(128),               -- 关联的业务 ID（可选）
    expire_at       BIGINT,                     -- 过期时间（临时文件用）

    PRIMARY KEY (id)
);

CREATE INDEX idx_file_meta_tenant ON p_file_meta (tenant_id, file_group, created_at);
CREATE INDEX idx_file_meta_biz ON p_file_meta (tenant_id, biz_type, biz_id);
CREATE INDEX idx_file_meta_md5 ON p_file_meta (tenant_id, md5);
```

## 3. API 设计

```
POST   /file/upload              上传文件 → 返回 fileId + 元数据
GET    /file/{fileId}            查询文件元数据（不下载）
GET    /file/{fileId}/url        获取访问 URL（public 直接返回，private 返回签名 URL）
GET    /file/{fileId}/download   下载文件内容
DELETE /file/{fileId}            删除文件（软删除元数据 + 异步清理存储）
POST   /file/batch-query         批量查询元数据（传 fileId 列表）
```

## 5. 业务方使用方式

```java
// 上传头像 → 只存 fileId
Long avatarFileId = fileCenterService.upload(file, tenantId, userId, "image", "avatar");
user.setAvatarFileId(avatarFileId);

// 显示头像 → 用 fileId 获取 URL
String avatarUrl = fileCenterService.getAccessUrl(avatarFileId);

// Excel 导出 → 结果文件存 fileId
Long resultFileId = fileCenterService.upload(excelFile, tenantId, userId, "temp", "excel-export");
task.setResultFileId(resultFileId);
```
